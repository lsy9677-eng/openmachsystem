# 테스트 체크리스트

1. `main-v2.html` 접속
2. `새 본선 추첨` 실행
3. 64강 32경기가 생성되는지 확인
4. `본선 코트배정` 실행
5. 각 코트에 시합중 1 + 대기1 1만 있는지 확인
6. 나머지가 공용대기인지 확인
7. 시합중 경기 결과를 6:1로 입력
8. 실제 점수 6:1이 대진표와 현황에 표시되는지 확인
9. 기존 대기1이 시합중으로 올라가는지 확인
10. 공용대기 1순위가 해당 코트의 대기1로 이동하는지 확인
11. 공용대기 카드가 시합중으로 직접 가지 않는지 확인
12. 두 인접 64강 경기 승자가 같은 32강 카드에 들어가는지 확인
13. 새로 확정된 32강 경기가 공용대기에 들어가는지 확인
14. JSON 내보내기/가져오기 확인
15. 새로고침 후 상태 유지 확인
