# 읽기 전용 연결 테스트

1. main-v2.html?v=4 접속
2. `기존 앱 새 창 열기`
3. 열린 dev.html에서 로그인
4. 테스트 대회와 부서 선택
5. V2 창으로 돌아와 `열린 앱 데이터 검사`
6. 후보 목록에서 팀 수와 샘플 확인
7. 순위 데이터라면 `조별 1·2위 명단 만들기`
8. 명단 입력창에서 이름과 팀 수 확인
9. `실제 명단 적용`
10. 새 본선 추첨 및 코트배정

후보가 나오지 않거나 잘못된 후보만 나오면 `진단 JSON 저장` 후 해당 파일을 전달하면
정확한 Firebase/상태 경로용 어댑터를 만들 수 있습니다.
