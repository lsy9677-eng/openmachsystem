# V1.10.1 Firebase 인증 세션 테스트

1. 기존 앱에서 관리자 또는 승인된 운영자로 로그인합니다.
2. 기존 앱 탭을 닫지 않습니다.
3. V2를 새로고침합니다.
4. `기존 앱 설정 탐색`
5. `로그인 세션 연결`
6. 인증 사용자 이메일 또는 UID가 표시되는지 확인합니다.
7. `연결 검사`
8. `현재 상태 테스트 저장`
9. `목록 새로고침`
10. 저장된 스냅샷을 선택해 JSON 저장 및 V2 상태 복원을 검사합니다.

## 결과 구분
- 로그인 필요: 같은 브라우저에 Firebase 로그인 세션 없음
- 운영자 권한 확인 필요: 로그인은 됐지만 users.approved 또는 admins 권한 부족
- Firebase 인증 연결: 인증과 전용 컬렉션 접근 모두 정상
