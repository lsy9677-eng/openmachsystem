# V1.12 운영 반영 준비 테스트

1. 기존 앱 연결 후 대회·부서를 선택합니다.
2. 확정 진출팀을 V2에 적용합니다.
3. 본선 추첨·코트 배정·결과 입력을 진행합니다.
4. `현재 연결 대상으로 고정`을 누릅니다.
5. `동기화 계획 만들기`를 누릅니다.
6. 완료·시합중·대기1·공용대기 수가 실제 화면과 같은지 확인합니다.
7. `안전성 검사`를 실행합니다.
8. 대진 잠금이 해제라면 경고가 표시되는지 확인합니다.
9. `반영 전 복구점 저장`을 실행합니다.
10. `운영 반영 패키지 저장`으로 JSON을 내려받습니다.
11. 기존 앱 데이터가 전혀 변경되지 않았는지 확인합니다.

이 단계는 쓰기 전용 사전검증 단계입니다.
