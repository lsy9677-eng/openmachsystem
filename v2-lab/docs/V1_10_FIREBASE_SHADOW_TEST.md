# V1.10 Firebase 별도 테스트 저장소 테스트

1. 기존 앱 새 창을 열고 로그인합니다.
2. V2에서 실제 대회·부서 진출팀을 적용합니다.
3. `기존 앱 설정 탐색`을 누릅니다.
4. Project ID와 API Key가 채워졌는지 확인합니다.
5. 전용 컬렉션 이름이 `mainV2ShadowTests`인지 확인합니다.
6. `연결 검사`를 누릅니다.
7. 성공하면 `현재 상태 테스트 저장`을 누릅니다.
8. 목록 새로고침 후 저장된 문서를 선택합니다.
9. JSON 저장과 V2 화면 복원을 각각 시험합니다.
10. Firebase Console에서 기존 운영 컬렉션이 변경되지 않았는지 확인합니다.

## 규칙에 의해 차단되는 경우
Firebase 저장이 실패해도 브라우저 백업 저장은 정상 작동합니다.
이 경우 다음 단계에서 전용 테스트 컬렉션에만 허용되는 보안 규칙을 별도로 설계합니다.
