# v1.4 테스트 순서

## 100팀 테스트
1. `100팀 테스트 명단 불러오기`
2. 대진 규모가 128강으로 바뀌는지 확인
3. `실제 명단 적용`
4. 부전승이 28자리인지 확인
5. `본선 코트배정`
6. 실제 경기만 시합중/대기1/공용대기에 들어가는지 확인
7. 결과 입력 후 64강 진출을 확인

## 파일 가져오기
지원 형식:
- TXT: 한 줄에 한 팀
- CSV: 선수1,선수2,소속
- JSON: {"teams":[{"name":"선수1 / 선수2"}]}

XLSX 파일은 브라우저에서 직접 읽지 않습니다. XLSX는 TXT/CSV/JSON으로 변환하여 사용합니다.
