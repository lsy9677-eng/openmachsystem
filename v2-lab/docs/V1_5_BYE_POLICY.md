# 부전승 배정 정책

100팀을 128강으로 운영하면 부전승은 28개입니다.

기본 정책:
1. 명단 입력 순서는 각 조의 `1위, 2위` 반복
2. 28개 부전승은 조 1위 50팀 중에서만 우선 선정
3. 1조부터 50조까지 특정 구간에 몰리지 않도록 균등 선정
4. 128강 64경기 전체에도 부전승 경기가 균등하게 분포
5. 부전승을 받지 못한 조 1위는 다른 조 2위와 우선 대결
6. 같은 조의 1위·2위는 첫 경기에서 만나지 않음

주의:
실제 명단이 `조1위, 조2위` 반복 순서가 아닌 경우
‘전체 팀 균등 무작위’를 선택해야 합니다.
