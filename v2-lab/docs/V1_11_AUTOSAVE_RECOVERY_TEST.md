# V1.11 자동저장·복구 테스트

1. 기존 앱 로그인 세션과 Firebase 연결을 확인합니다.
2. 실제 64팀 명단을 V2에 적용합니다.
3. 본선 추첨과 코트 배정을 실행합니다.
4. `자동저장 ON`을 누릅니다.
5. 경기 결과 1건을 입력합니다.
6. 약 3초 후 브라우저·Firebase 저장 시간이 갱신되는지 확인합니다.
7. 경기 결과를 추가로 입력하고 다시 저장 시간이 갱신되는지 확인합니다.
8. `복구점 새로고침` 후 여러 복구점이 보이는지 확인합니다.
9. 이전 복구점을 선택해 미리보기와 JSON 저장을 확인합니다.
10. `선택 복구` 후 대진·코트·점수가 해당 시점으로 돌아가는지 확인합니다.
11. `현재 상태 검증` 결과가 통과인지 확인합니다.

## 주의
자동저장은 V2 상태만 저장합니다.
기존 앱의 운영 문서에는 자동으로 반영하지 않습니다.
