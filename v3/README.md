## 현재 포함
- JSON 명단 불러오기
- 32/64/128강 대진 생성
- 조 1위 우선 배치 기반 슬롯 구성
- 코트별 시합중 1 + 대기1 1
- 공용대기
- 결과 입력
- 승자 다음 라운드 반영
- 코트 큐 자동 승계
- 브라우저 자동 저장
- 수동 복구점
- 운영 로그
- 반응형 새 CSS
