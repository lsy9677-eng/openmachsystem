# 230MATCH V3 — Stage 20 Main Draw Balanced Assignment

수정:
- 예선 코트배정은 기존처럼 1조, 2조, 3조 순서대로 순환 배정
- 본선 추첨 완료 후 경기번호를 상단·하단·중앙 순서로 재정렬
- 예: 1경기, 마지막 경기, 중앙 경기, 2경기, 마지막-1경기, 중앙 인접 경기...
- 국제·원도심 등 구장별 코트 수에 비례해 본선 경기 분배
- 각 구장 안에서 시합중·대기1·공용대기를 균등 구성
- 새로 준비되는 다음 라운드 경기에도 동일한 균등 순서 적용
- 가지형 대진표·수동대기·코트상태 기능 유지
