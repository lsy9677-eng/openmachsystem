# 230MATCH V3 — Stage 20.3 Prelim Dynamic Order

수정:
- 예선 조편성 화면에서 대회 설정의 예선 사용 구장·총 코트 수 표시
- 3팀조 경기 순서:
  1. 1번팀 vs 2번팀
  2. 첫 경기 승자 vs 3번팀
  3. 첫 경기 패자 vs 3번팀
- 2·3경기는 앞 경기 결과에 따라 팀이 동적으로 확정
- 한 코트에 여러 조가 배정되면 조별 첫 경기를 번갈아 운영
- 첫 경기 완료 시 해당 조의 두 번째 경기를 코트 대기열 뒤에 추가
- 두 번째 경기 완료 시 세 번째 경기를 대기열 뒤에 추가
- 각 예선 코트 카드에 조 번호·경기 번호 표시
- 조별 카드에 구장명·코트명 표시
