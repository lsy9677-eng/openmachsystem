# 230MATCH V3 — Stage 3 Prelim → Draw

기존 앱과 완전히 분리된 독립 운영본입니다.

## 3단계 추가
- 예선 진행 중 본선 선추첨
- 본선 슬롯을 `1조 1위`, `1조 2위` 형태로 미리 배치
- 예선 조 순위 확정 시 실제 팀명 자동 치환
- 본선 진행 전 수동 동기화
- 이미 시합중·완료된 경기는 자동 치환 금지
- 연결 상태 카드로 대기·완료·잠금 표시

## 안전 규칙
- 본선 시합중·완료 경기의 팀 슬롯은 자동 변경하지 않습니다.
- 예선 조편성을 새로 생성하면 기존 연결 대진 정보는 초기화됩니다.
- Firebase와 기존 앱 코드는 사용하지 않습니다.
